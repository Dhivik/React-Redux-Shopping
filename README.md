# 🛒 React E-Commerce Store (FakeStore API)

## 📖 Description
This project is a frontend e-commerce application built using React. It allows users to browse products, add items to cart and wishlist, and manage cart data with persistence using local storage.

The application demonstrates core frontend concepts such as state management, API integration, and reusable component architecture.

---

## 🚀 Features
- Display products using FakeStore API
- Add items to Cart and Wishlist
- Dynamic cart quantity management
- Increase quantity when adding the same product
- Decrease quantity with automatic removal when quantity reaches zero
- Total cart price calculation
- Cart persistence using Local Storage
- Responsive UI

---

## ⚙️ Tech Stack
- React.js

- CSS3
- FakeStore API

---

## 🧠 Concepts Covered
- React Hooks (useSelector, useEffect)
- API Integration
- State Management
- Component Reusability
- Conditional Rendering
- Event Handling
- Local Storage Handling

---

## 🛒 Cart Functionality
- Items added to cart are stored in local storage
- Cart data persists even after page refresh
- Adding the same product increases quantity dynamically
- Users can increase or decrease item quantity
- If item quantity reaches 0, it is automatically removed from the cart
- Displays total cart price based on selected items
